# FE_DuAnKhoiNghiep

Frontend React + TypeScript cho dự án khởi nghiệp.

## Yêu cầu

- Node.js 20 trở lên
- npm

Kiểm tra phiên bản:

```bash
node -v
npm -v
```

## Clone project

```bash
git clone https://github.com/NgoThiMinhQuang/FE_DuAnKhoiNghiep.git
cd FE_DuAnKhoiNghiep
```

## Cài thư viện

```bash
npm install
```

## Chạy project

```bash
npm run dev
```

Sau khi chạy, mở đường dẫn Vite hiển thị trên terminal, thường là:

```text
http://localhost:5173/
```

## Build production

```bash
npm run build
```

## Xem bản build

```bash
npm run preview
```

## Scripts

- `npm run dev`: chạy môi trường development
- `npm run build`: kiểm tra TypeScript và build production
- `npm run preview`: xem thử bản production build
- `npm run lint`: kiểm tra code bằng Oxlint
